Chat Buddy — Embeddable Widget Test

This page demonstrates embedding the Chat Buddy widget via an iframe. Replace the LECTURE_ID in the src URL with a real lecture ID and update the domain to your deployment URL.

Embed Code

<iframe
  src="http://localhost:3000/embed/YOUR_LECTURE_ID"
  width="400"
  height="620"
  style="border:none; border-radius:12px;"
  allow="clipboard-write"
></iframe>
